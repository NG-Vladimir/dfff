# График служения

PWA для планирования служения — назначение музыкантов на вторники и воскресенья.

**Стек:** React + Vite + Supabase + PWA + Vercel + Telegram Bot

## Возможности

- **Люди** — добавление и удаление музыкантов
- **Роли** — ведущий, гитарист, клавишный, барабанщик, бэк вокал
- **Выбор дня** — календарь вторников и воскресений по месяцам
- **Назначения** — выбор человека на каждую роль для выбранного дня
- **График на неделю** — текущая неделя (ближайшие вторник и воскресенье)
- **График за месяц** — все вторники и воскресенья текущего месяца
- **Отправить в Telegram** — формирование сообщения и отправка через Share API

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Настройка

- **Supabase** — см. [SUPABASE-SETUP.md](SUPABASE-SETUP.md) (облачное хранение)
- **Telegram** — см. [SETUP-TELEGRAM.md](SETUP-TELEGRAM.md) (автоотправка в группу)

Без Supabase данные хранятся в localStorage. Без Telegram — копирование в буфер + открытие группы.

## PWA

Приложение можно установить на телефон или компьютер («Добавить на главный экран» / «Install» в браузере).
